📋 Cara Kerja

1 — Tambah mod lewat menu Tambah Mod

2 — Data tersimpan di browser (localStorage)

3 — Pergi ke Generate Kode JS

4 — Copy kode yang dihasilkan

5 — Paste ke script.js di VS Code

6 — Taruh foto ke folder foto/

7 — Taruh file RAR/ZIP ke folder skin/

8 — Deploy ulang website → selesai ✓
