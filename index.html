Cara Install Mod

01

Download File

Klik tombol download pada mod yang kamu inginkan. File akan terunduh dalam format .rar atau .zip.

02

Extract File

Extract menggunakan WinRAR atau 7-Zip. Backup folder GTA SA kamu terlebih dahulu sebelum install.

03

Install Skin

Untuk skin (.txd/.dff): Buka IMG Tool v2, import file ke player.img di folder GTA SA.

04

Install Animasi

Untuk pose/animasi (.ifp/.cs): Salin ke folder anim/ atau CLEO/. Butuh CLEO Library 4.
